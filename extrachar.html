1: Duplicating the character sprite

self explanatory

Obviously just go into the character sprite and duplicate a sprite that doesn't have any sort of animation on it brah

2: The process

the part that makes the character alive in general

First of all you're going to go into the NEW duplicated character sprite and then change the part at the character properties block in the new sprite that says "Number this character identifies as"

Change that part inside of the character properties block to the number of the character

Then after this, rename the sprite to your character name you are planning to do (Recommended if you added your first extra character and is sticking with the default name: S1 (Special 1)

Go into the Initializer folder and then click on the "variables" sprite inside of it and then edit this part to the amount of characters you have in your project (normally 20)

3: The icon process

Making the character placable

To do this, first you go into the "icons" sprite (located in UI/Essentials)

Duplicate a already existing costume (recommended) and then rename it to the exact name you named your extra character (MAKE SURE THE EXTRA CHARACTER IS IN THE CHARACTERS FOLDER!!! DON'T MAKE A NEW FOLDER)

Now that you've done the costume part, we can now get to the code part

At this part inside of the icon code right here, make a new repeat block duplicated from one of the default repeat initializers being repeat 1 (or the amount of characters you have in the extra icon section)

Now, you should be done!
